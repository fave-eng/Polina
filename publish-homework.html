Публикация домашки

Открывает урок в Supabase и отправляет уведомление в Telegram один раз.

Важно: эта страница использует Telegram Bot API прямо из браузера. Для личного сайта это быстро и удобно, но токен бота виден в коде. Более безопасный вариант — Supabase Edge Function.

Сейчас стоит chat_id из твоих страниц. Если нужно писать Полине лично, сюда нужно вставить её личный chat_id.
